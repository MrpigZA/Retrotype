# **App Name**: RetroType

## Core Features:

- Text Input: Enable users to input text via a text editor with a vintage typewriter aesthetic and optional clicky sound effects.
- Document Saving: Automatically save the current document every 5 minutes in the background to prevent data loss.
- Open saved files: Open files previously saved. Users are shown a list of available files, each of which can be selected for viewing or editing.

## Style Guidelines:

- Primary color: Muted cream (#F5F5DC) for a vintage paper effect.
- Background color: Very light cream (#FDFEF8). It should be visibly of the same hue as the primary color, but heavily desaturated and brighter to ensure it adequatelly serves as a background.
- Accent color: Deep charcoal (#36454F) for text and UI elements to contrast with the cream background.
- Headline font: 'Literata', a serif font with a literary, vintage, slightly formal feel.
- Body font: 'Literata', a serif font with a literary, vintage, slightly formal feel.
- Use simple, classic icons for file operations like save, open, and new document.
- Implement typewriter-like cursor animation.
